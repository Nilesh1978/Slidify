Shiny App Description

  1. Name of the App is "diversity""
  2. It is publicly avaiable app
  3. This app allows you to dynamically get similarity values by year

Tab Description

This app has three tabs.

  1. Data description: Provides description of the data
  2. Data: Gives data summary
  3. Plot: plots histogram for similarity values by the selected year

Similarity values by year

diversity<- read.csv("project.csv", stringsAsFactors = FALSE)
diversity$CrossYear<- as.factor(diversity$CrossYear)
tapply(diversity$Similarity, diversity$CrossYear, mean, na.rm=T)
